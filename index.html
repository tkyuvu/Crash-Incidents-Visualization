This project aims to analyze crash incidents data using exploratory analysis, provide visualizations, conduct causal research question and predictive question, and build statistical models to answer the quetions.

The causal research question is: Is the variable 'Hit/Run' dependent on the variable 'Weather'?
The predictive research question is: Will a person more likely to hit and run in clear weather?

Causal Research Question

The question is "Is Hit/Run dependent on the Weather?"
The null hypothesis: Hit/Run and Weather are independent.
The alternative hypothesis: Hit/Run and Weather are dependent on each other.

To answer this question, Chi-square independence test is used.
The columns ‘Hit/Run’ and ‘Weather’ are both categorical, so the contingency table is created.


Chi-square of independence test is performed and here is the result


The p-values are much less than 0.05, so we reject the null hypothesis. That means the variables ‘Hit/Run’ and ‘Weather’ are dependent of each other.

Predictive Research Question

The question is "Will a person more likely to hit and run when the weather is clear?"
Null hypothesis: The chance that a person will hit and run is the same for different weather conditions.
Alternative hypothesis: The chance that a person will hit and run is not the same for different weather conditions.


To answer the question, Logistic Regression is used, and the p-values and odd-ratios will be used.
The target for the model is ‘Hit/Run’. The dataset is split into 80% training set and 20% testing set. The evaluation metrics are F1 score and ROC curve.
Confusion matrix


F1 score of the model is 70.4% and ROC curve metric is 91.5%. This is a good model.
Because the Weather and Hit/Run variables are the two main variables in the question, I will obtain the p-values and odd-ratios from the model for only weather conditions. That is what I will focus on to answer the question.


The p-value greater than 0.05 means that the feature is not significant, so the significant weather conditions are Blowing Snow, Clear, Cloudy, Raining, and Wintry Mix. The odd ratios can tell the percent chance an event is more (or less) likely to happen. Looking at the odd ratios, we can interpret the result as follow:
In clear weather, hit and run is 1.1 times more likely to happen. There are 6% less likely to hit and run in Blowing Snow, 2% less likely to hit and run in cloudy weather, 4% less likely to hit and run in the rain, and there's no difference in wintry mix (because odd ratio for wintry mix is 1.0).


To answer the hypothesis:
Based on the results, we reject the null hypothesis and can make conclusion that the chance a person will hit and run is not the same for different weather conditions.
A person is 1.1 times more likely to hit and run in clear weather.

Conclusion

1. For causal question, Hit/Run and Weather variables are dependent of each other.

2. For predictive quesiton, a person is 1.1 times more likely to hit and run in clear weather. The chance for hit and run is different in different weather conditions.
